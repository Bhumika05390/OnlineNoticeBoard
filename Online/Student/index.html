Online Notice Board

Online Notice Board System is a project that can manage and display notices for students in colleges.The program was developed by using PHP,MySQL,and javascript. The system is very easy to use,the user must first creste a new account before accessing the system.The user can view the notice information from theirrespective adviser or staff. The admin is the one and only that can put up a notice to each student on the campus.
